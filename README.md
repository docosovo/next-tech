# Next Tech Mech DOOM

Sitio web desarrollado para la promoción y comercialización de productos tecnologicos, diseñado con una interfaz moderna, organizada y facil de usar, la plataforma busca ofrercer una experiencia digital para los usuarios incluyendo un carrito de compras donde se añaden los productos orientando a la presentación de manera clara y consiza, de igual manera tiene tanto inicio de sesion como registro, ademas de incorporar una barra de busqueda para productos de la pagina y si no se encuentras en la base de datos de la pagina, el usuario sera redireccionado a una busqueda por fuera de la pagina usando su navegador

## Tecnologias usadas

- HTML
- CSS
- JavaScript
- PHP
